# Data Analytics in Sales Domain

🔍 Project Overview

This project focuses on applying data analytics techniques in the sales domain to uncover meaningful insights and support data-driven decision-making. It demonstrates skills in data cleaning, analysis, and visualization using real-world datasets.

🎯 Objectives
Analyze sales data to identify trends and patterns
Perform data cleaning and transformation
Generate actionable business insights
Build interactive dashboards for better visualization
🛠️ Tools & Technologies
Python (Pandas, NumPy)
SQL (Data extraction & queries)
Excel (Data preprocessing)
Power BI / Tableau (Data visualization)


📁 Dataset
The dataset is provided in the repository in Excel format
It includes sales-related data such as:
Orders
Revenue
Customers
Products


📊 Key Analysis Performed
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Sales Trend Analysis
KPI Metrics (Revenue, Profit, Growth)
Customer & Product Insights


📈 Visualizations
Interactive dashboards
Sales performance charts
Trend analysis graphs
